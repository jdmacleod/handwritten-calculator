Handwritten Calculation Recognition demo

Prediction of simple handwritten integer calculations using Digit and Symbol image classification.

Uses TensorFlow models trained on MNIST dataset for digits and provides a dataset for symbols, along with an example capture and processing pipeline that was used to collect the symbol dataset.

Source: https://github.com/jdmacleod/handwritten-calculator
